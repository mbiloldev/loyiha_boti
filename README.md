# 📈 AI Trading Forecast & Signals Telegram Bot

Ushbu Telegram bot traderlar uchun real vaqt rejimidagi (`real-time`) bozor tahlillari, grafik prognozlar va aniq **BUY / SELL** signallarini taqdim etish uchun ishlab chiqilgan. Bot nafaqat joriy holatni baholaydi, balki o'tmishdagi tahlillar arxivini ham foydalanuvchiga yubora oladi.

## ✨ Asosiy Funksiyalar (Features)

* **⚡ Real-Time Price Tracking:** Bozor narxlari ritmini va o'zgarishlarini soniyalarda kuzatib borish.
* **📊 Chart Generation:** Texnik tahlil va prognoz ko'rsatilgan vizual grafik (rasm) ko'rinishidagi hisobotlarni yuborish.
* **🚨 Trading Signals:** Aniq strategiyaga asoslangan **BUY (Sotib olish)** yoki **SELL (Sotish)** signallari.
* **🗄️ Forecast Archive:** Oldindan qilingan prognozlar va ularning natijalari aks etgan rasmlar bazasi.

---

## 🛠️ Texnologiyalar (Tech Stack)

* **Python 3.x** - Asosiy dasturlash tili.
* **Aiogram (yoki Telebot)** - Telegram Bot API bilan ishlash uchun.
* **Pandas & NumPy** - Ma'lumotlarni qayta ishlash.
* **Matplotlib / Plotly** - Grafik va prognozlarni chizish (Rasm ko'rinishida).
* **Requests / WebSockets** - Birjalardan real-time ma'lumotlarni olish (API).
